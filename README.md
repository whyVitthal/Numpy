# 🔢 NumPy Learning Repository

A simple reference guide and practice workspace for mastering NumPy arrays.

---

## 📂 Topics Covered

This repository contains the following notebooks:

1. **`arrays.ipynb`**
   * Introduction to NumPy arrays, array creation, and basic attributes.

2. **`Array_Indexing.ipynb`**
   * Slicing multi-dimensional arrays, extracting specific rows/columns, and boolean filtering.

3. **`Array_Operations.ipynb`**
   * Vectorized math operations, arithmetic calculations without loops, and array aggregations.

4. **`exersises.ipynb`**
   * Practice problems, matrix manipulation challenges, and code optimization exercises.
